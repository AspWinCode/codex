# Кодэкс — Агентство цифровых расследований

Детективная образовательная платформа. Cinematic onboarding, кейсы, лаборатория кода.

## Стек
- Frontend: React 18 + TypeScript + Vite + Tailwind
- Backend: FastAPI + PostgreSQL + Redis
- Deploy: Docker
